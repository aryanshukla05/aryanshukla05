## Featured Projects

### 1. Electric Output for Testing Electromagnetic Effects on Microalgae Growth

**Topics:** embedded systems, electronics prototyping, bioengineering experimentation, Arduino, Teensy, CAD, 3D printing

Developed a modular experimental platform to study how electric and magnetic fields influence microalgae growth. The system combined multi-channel PWM control, custom electronics, and 3D-printed mechanical structures for long-duration experiments.

**Highlights**

* Built a microcontroller-based multi-channel control system
* Designed modular flask fixtures and electrode positioning hardware
* Implemented EEPROM-based parameter storage for autonomous runs
* Balanced thermal, electrical, and mechanical constraints in a real lab setup

### 2. Multimodal Deforestation Detection from Satellite Data

**Topics:** machine learning, remote sensing, geospatial systems, Python, LightGBM, MLP, Sentinel data

Built an end-to-end machine learning pipeline for pixel-level deforestation detection using multimodal satellite imagery and temporal embeddings during the TUM.ai Makeathon 2026.

**Highlights**

* Combined radar, optical, and embedding-based features
* Trained and compared multiple models including LightGBM and MLP
* Generated GeoJSON outputs for spatial evaluation
* Focused on robustness under noisy labels and regional domain shift

### 3. Mobile Computer Vision for Indoor Localization

**Topics:** computer vision, TensorFlow, Keras, MobileNetV2, mobile ML

Developed a lightweight indoor location classifier for the TUM Ottobrunn campus using mobile images. The project compared CNNs trained from scratch against transfer learning approaches for efficient deployment.

**Highlights**

* Built a MobileNet-based image classification pipeline
* Collected and processed a real-world student-generated dataset
* Applied transfer learning and augmentation to improve generalization
* Created a web-based interface for prediction
